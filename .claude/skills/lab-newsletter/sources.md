# Newsletter information sources ("channels" the digest gathers from)

The nightly digest pulls from these channels. Use `WebSearch`/`WebFetch` (and the
`deep-research` skill in prod) against them; prefer primary sources; cite everything.
This list is the lab's **information radar** — keep it pruned and current as strategy
shifts (it's part of "keep reshaping our strategy").

## A. The horizon — field & future
**Preprints / papers**
- arXiv: `cs.AI`, `cs.LG`, `cs.MA` (agents), `q-bio.QM`, `q-bio.CB` — e.g. search
  `site:arxiv.org <topic> 2026`.
- bioRxiv / medRxiv: bioinformatics, systems biology, cell biology, synthetic biology.
- Journals: *Nature Methods*, *Nature Biotechnology*, *Nature Machine Intelligence*,
  *Cell Systems*, *Cell*, *Molecular Systems Biology*, *Bioinformatics*.
**Virtual cell / automated discovery**
- CZI **Virtual Cells** Platform; Arc Institute **Virtual Cell Atlas**; Chan-Zuckerberg, DeepMind Isomorphic.
- "AI scientist" / self-driving-lab work (Sakana AI, autonomous-lab groups).
**Advanced AI tools & development**
- Lab blogs: DeepMind, Anthropic, OpenAI, Google Research, Meta FAIR, Hugging Face, Mistral.
- Agent frameworks & tooling releases; foundation-model releases (incl. for biology: ESM, AlphaFold, scGPT/scFoundation lineage).
**State-of-the-art cell biology & omics**
- Single-cell / spatial / multi-omics methods; imaging & microscopy advances.
